## 🚀 Getting Started

### 기본 설정

```
git clone https://github.com/bassyu/react-payments.git
cd javascript-movie-review
```

### step1

```
git checkout -b step1
git pull origin step1
npm ci
npm run start
```

```
npm ci
npm run start
```

```
npm run storybook
```
